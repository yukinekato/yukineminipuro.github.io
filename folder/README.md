# yukineminipuro
====
カラス撃退ゲーム


## Description　
ガリバー行け（鴨池）にいる鴨を狙うカラスを撃退することで鴨を守るゲーム いわゆるクソゲーだが少し難しくすることで何回もやりたくなるようにする。

## VS.

## Requirement
・タイマー機能を使う（60秒で「あと30秒」９０秒で「タイムアウト」と表示） ・鴨池と鴨とカラスはイラストor写真
